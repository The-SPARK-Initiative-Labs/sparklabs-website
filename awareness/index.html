This is called an "Evil Twin" or "Rogue Access Point" attack. Here's how it works:

  1. Attacker creates a WiFi network with a trusted name (like "Starbucks WiFi" or "Hotel Guest")
  2. Your device connects ‚Äî sometimes automatically if you've used that name before
  3. A fake login page appears, looking legitimate
  4. You enter credentials thinking you're logging into WiFi or a service
  5. The attacker now has your username and password
